# 프로그래밍의 절차

유명한 명언이야기.

1. 우선 동작하게 하라.(Make it work)
2. 코드를 다시 자료구조, 알고리즘을 고려하라.(여러번 수정하고 짜봐야한다.) Make it right
3. 속도와 성능을 올려라. Make it fast

2번3번을 잘 하기 위해서..
수학 물리, 프로그래밍 언어 자료구조 알고리즘 OS, HW(CPU, Memory)
